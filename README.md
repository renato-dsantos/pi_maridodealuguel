# 🏠 Marrão de Aluguel

> Projeto de conclusão do **3º semestre** do curso de Desenvolvimento Web.

---

## 📋 Sobre o Projeto

O **Marrão de Aluguel** é uma plataforma web que conecta **prestadores de serviços** — como pedreiros, eletricistas e encanadores — a **pessoas que precisam contratar esses profissionais**.

O sistema permite que prestadores se cadastrem informando seus serviços e que clientes encontrem o profissional ideal de forma rápida e prática.

---

## ✨ Funcionalidades

- Cadastro de prestadores de serviço
- Login de usuários (clientes e prestadores)
- Seleção de especialidade (Eletricista, Encanador, Pedreiro, Pintor, Jardineiro, Montador de Móveis)
- Buscar pretadores pela especialidade
- Conexão entre cliente e prestador
- Avaliação de serviço
